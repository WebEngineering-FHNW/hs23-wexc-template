automatically enforced

automatically enforced

otherwise invalid

otherwise invalid

automatically enforced

Composing Projectors.

We go from simple to more advanced UI elements by combining previous work without touching previous functionality. Projectors (view and binding) might need slight modifications.

Again, you can replace the currently simple projector as used in starter.js with your own view and binding by only changing the import file name and nothing else!

BTW: it is ok to make stylistic and layout changes to this HTML file and the CSS to accommodate your needs.
